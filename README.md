# 🌡️ Temperature Converter

A simple and beginner-friendly web app that converts temperatures between **Celsius and Fahrenheit**. Built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- Convert Celsius → Fahrenheit
- Convert Fahrenheit → Celsius
- Simple and clean UI
- Instant result display
- Input validation for unit selection

---

## 🛠️ Built With

- HTML
- CSS
- JavaScript (Vanilla)

---


---

## ⚙️ How It Works

1. Enter a temperature value in the input box
2. Select a conversion type:
   - Celsius → Fahrenheit
   - Fahrenheit → Celsius
3. Click **Submit**
4. Get the converted result instantly

---

## 📌 Formula Used

- Celsius → Fahrenheit  
  `F = (C × 9/5) + 32`

- Fahrenheit → Celsius  
  `C = (F - 32) × 5/9`
